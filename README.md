# LaunchStack - Sample Generated Output

This repository contains a **sample project generated by LaunchStack**, an Azure infrastructure generation wizard.

> **Note:** This repository is **not the source code** of LaunchStack. It is a sample of the generated output so DevOps engineers can inspect the repository structure, Terraform modules, Helm charts, CI/CD pipelines, and overall project organization.

---

## What is LaunchStack?

LaunchStack is a guided wizard that generates production-ready Azure infrastructure repositories.

Instead of starting from scratch, you answer a few questions about your architecture and LaunchStack generates a complete repository containing:

- Terraform modules
- Helm charts
- GitHub Actions / Azure DevOps pipelines
- Monitoring configuration
- Documentation
- Deployment scripts

Current supported deployment targets include:

- AKS
- Azure App Service
- Azure Container Apps
- Azure Functions
- Static Web Apps
- Azure Storage
- Azure SQL
- PostgreSQL
- Azure Key Vault
- Azure Container Registry
- Virtual Network
- Application Gateway
- Front Door
- Azure Monitor

## Why this repository exists

Several members of the Reddit Terraform community asked to see the generated output before trying the wizard.

This repository provides a preview of:

- Folder structure
- Generated templates
- Project organization
- Infrastructure layout

without requiring anyone to run the wizard first.
